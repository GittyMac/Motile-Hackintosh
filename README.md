# Motile M141/M142 Hackintosh
This is the EFI for the Walmart MOTILE M141/M142 (PF4PU1F) Ryzen laptops. This should also work for other Ryzen PF4-F Tongfang laptops.

## Compatibility

 - [x] Video
 - [x] Keyboard
 - [x] Audio
 - [x] Wi-Fi
 - [x] Bluetooth
 - [x] Touchpad (Polling)
 - [ ] GPIO Touchpad
 - [ ] Sleep
 - [ ] Microphone
